# cc-toolkit

A Claude Code plugin marketplace with productivity tools.

## Installation

Add the marketplace:

```
/plugin marketplace add grixu/cc-toolkit
```

## Available Plugins

### codex-plan-improver

Automatically reviews implementation plans via OpenAI Codex CLI before exiting plan mode. Claude and Codex go back-and-forth until the plan is approved.

```
/plugin install codex-plan-improver@cc-toolkit
```

See [plugin README](plugins/codex-plan-improver/README.md) for details.
